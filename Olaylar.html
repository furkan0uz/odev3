JavaScript Olayları

HTML olayları, HTML öğelerinde meydana gelen "şeyler" dir . HTML sayfalarında JavaScript kullanıldığında, JavaScript bu olaylara "tepki verebilir" .

HTML Etkinlikleri

HTML etkinliği, tarayıcının yaptığı veya kullanıcının yaptığı bir şey olabilir. HTML olaylarına bazı örnekler:

  • Bir HTML web sayfasının yüklenmesi tamamlandı
  • Bir HTML giriş alanı değiştirildi
  • Bir HTML düğmesi tıklandı

    • Çoğu zaman, olaylar meydana geldiğinde, bir şeyler yapmak isteyebilirsiniz. JavaScript, olaylar algılandığında kod yürütmenizi sağlar. HTML, HTML kodlarına JavaScript kodlu olay işleyici niteliklerinin eklenmesine olanak tanır .

    Tek tırnak işaretleri ile:

    <element event='some JavaScript'>

    Çift tırnak işareti ile:

    <element event="some JavaScript">

    Aşağıdaki örnekte, onclick bir 

     <button>
    öğeye (kodlu) bir öznitelik eklenmiştir :

                    <button 
                onclick="document.getElementById('demo').innerHTML=Date()">
                The time is?</button>

    Ortak HTML Etkinlikleri

    Aşağıda bazı yaygın HTML etkinliklerinin küçük bir kısmının listesi verilmiştir:

    Etkinlik Açıklama
    Onchange Bir HTML öğesi değiştirildi
    tıklamada Kullanıcı bir HTML öğesini tıklar
    onmouseover Kullanıcı fareyi bir HTML öğesinin üzerine taşır
    onmouseout Kullanıcı fareyi bir HTML öğesinden uzaklaştırır
    onkeydown Kullanıcı bir klavye tuşuna basar
    onload Tarayıcı sayfayı yüklemeyi tamamladı

    JavaScript ne yapabilir?

    Olay işleyicileri, kullanıcı girdisini, kullanıcı eylemlerini ve tarayıcı eylemlerini işlemek ve doğrulamak için kullanılabilir:

  • Bir sayfa her yüklendiğinde yapılması gerekenler
  • Sayfa kapalıyken yapılması gerekenler
  • Kullanıcı bir düğmeyi tıkladığında gerçekleştirilmesi gereken eylem
  • Bir kullanıcı veri girdiğinde doğrulanması gereken içerik

    • JavaScript'in etkinliklerle çalışmasına izin vermek için birçok farklı yöntem kullanılabilir:

  • HTML olay özellikleri doğrudan JavaScript kodunu yürütebilir
  • HTML olay özellikleri JavaScript işlevlerini çağırabilir
  • HTML öğelerine kendi olay işleyici işlevlerinizi atayabilirsiniz
  • Etkinliklerin gönderilmesini veya işlenmesini engelleyebilirsiniz
