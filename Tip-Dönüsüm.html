JavaScript Türü Dönüşümü

JavaScript'te değer içerebilen 5 farklı veri türü vardır:

  • string
  • number
  • boolean
  • object
  • function

    • 6 tür nesne vardır:

  • Object
  • Date
  • Array
  • String
  • Number
  • Boolean

    • Ve değerler içeremeyen 2 veri türü vardır:

  • null
  • undefined

    • Operatör Türü

    typeof: Bir JavaScript değişkeninin veri türünü bulmak için operatörü kullanabilirsiniz .

    Lütfen Dikkat:

  • NaN veri türü sayıdır
  • Bir dizinin veri türü nesnedir
  • Bir tarihin veri türü nesnedir
  • Null veri türü nesnedir
  • Undefined değişkeninin veri türü undefined *
  • Değer atanmamış bir değişkenin veri türü de tanımsızdır *
    •

    Typeof: Bir JavaScript nesnesinin bir dizi (veya tarih) olup olmadığını belirlemek için kullanamazsınız .

    Otomatik Tip Dönüştürme

    JavaScript "yanlış" bir veri türünde çalışmaya çalıştığında, değeri "doğru" bir türe dönüştürmeye çalışır.

    Sonuç her zaman beklediğiniz gibi değildir:

    5 + null // returns 5 because null is converted to 0
    "5" + null // returns "5null" because null is converted to "null"
    "5" + 2 // returns "52" because 2 is converted to "2"
    "5" - 2 // returns 3 because "5" is converted to 5
    "5" * "2" // returns 10 because "5" and "2" are converted to 5 and 2
