JavaScript Döngüler

JavaScript farklı döngü türlerini destekler:

  • for: bir kod bloğundan birkaç kez geçer
  • for/in: bir nesnenin özellikleri arasında dolaşır
  • for/of: yinelenebilir bir nesnenin değerleri arasında dolaşır
  • while: belirtilen koşul geçerliyken bir kod bloğundan geçer
  • do/while: belirli bir koşul geçerliyken bir kod bloğundan da geçer

    • Döngü

    for Döngü sözdizimi aşağıdaki gibidir:

                <script>
            var text = "";
            var i;
            for (i = 0; i < 5; i++) {
            text += "The number is " + i + "<br>";
            }
            document.getElementById("demo").innerHTML = text;
            </script>

    for döngü sözdizimi aşağıdaki gibidir:

  • İfade 1:Kod bloğunun yürütülmesinden önce (bir kez) yürütülür.(İ=0)
  • İfade 2:Kod bloğunu yürütme koşulunu tanımlar.(i<5)
  • İfade 3:Kod bloğu yürütüldükten sonra (her seferinde) yürütülür.(i++)

    • Bildirim 1

    Normalde döngüde kullanılan değişkeni başlatmak için 1. ifadeyi kullanırsınız (i = 0). Bu her zaman böyle değildir, JavaScript umursamaz. Bildirim 1 isteğe bağlıdır.

    Bildirim 2

    Genellikle ifade 2, ilk değişkenin durumunu değerlendirmek için kullanılır. Bu her zaman böyle değildir, JavaScript umursamaz. Bildirim 2 de isteğe bağlıdır. İfade 2 true değerini döndürürse, döngü yeniden başlar, false döndürürse döngü sona erer.

    NOT:2. ifadeyi atlarsanız , döngü içinde bir mola vermeniz gerekir . Aksi takdirde döngü asla bitmez. Bu, tarayıcınızı kilitler.

    Bildirim 3

    Genellikle ifade 3, ilk değişkenin değerini artırır. Bu her zaman geçerli değildir, JavaScript umursamaz ve ifade 3 isteğe bağlıdır. İfade 3, negatif artış (i--), pozitif artış (i = i + 15) veya başka bir şey yapabilir.

    For / In Döngüsü

    JavaScript for/in ifadesi bir nesnenin özellikleri arasında dolaşır:

                <script>
            var txt = "";
            var person = {fname:"John", lname:"Doe", age:25}; 
            var x;
            for (x in person) {
            txt += person[x] + " ";
            }
            document.getElementById("demo").innerHTML = txt;
            </script>

    Döngü İçin / Döngü

    JavaScript for/ofifadesi, yinelenebilir bir nesnenin değerleri arasında dolaşır.

    for/of Diziler, Dizeler, Haritalar, Düğüm Listeleri ve daha fazlası gibi yinelenebilir veri yapıları arasında geçiş yapmanızı sağlar.

    for/ofDöngü sözdizimi aşağıdaki gibidir:

                for (variable of iterable) {
            // code block to be executed
            }

    Değişken:Her yineleme için değişkene sonraki özelliğin değeri atanır. Değişken ile bildirilebilir const, let ya da var.

    İterable:Yinelenebilir özelliklere sahip bir nesne.

    While Döngüsü

    Döngü uzunluğunda belirli bir koşul doğru olarak kod bloğu döngüsü.

    Uygulama:

    Aşağıdaki örnekte, bir değişken (i) 10'dan az olduğu sürece döngüdeki kod tekrar tekrar çalışacaktır:

                <script>
            var text = "";
            var i = 0;
            while (i < 10) {
            text += "<br>The number is " + i;
            i++;
            }
            document.getElementById("demo").innerHTML = text;
            </script>

    NOT:Koşulda kullanılan değişkeni artırmayı unutursanız, döngü asla sona ermez. Bu, tarayıcınızı kilitler.

    Do / While Döngüsü

    Döngü sırasında döngünün bir çeşididir. Bu döngü, koşulun doğru olup olmadığını kontrol etmeden önce kod bloğunu bir kez yürütür, ardından koşul doğru olduğu sürece döngüyü tekrarlar.

    Uygulama:

    Aşağıdaki örnekte bir do/whiledöngü kullanılmıştır. Kod bloğu, koşul test edilmeden önce yürütüldüğünden, koşul yanlış olsa bile her zaman en az bir kez yürütülür: 

                <script>
            var text = ""
            var i = 0;
            do {
            text += "<br>The number is " + i;
            i++;
            }
            while (i < 10);  
            document.getElementById("demo").innerHTML = text;
            </script>

    NOT:Koşulda kullanılan değişkeni artırmayı unutmayın, aksi takdirde döngü asla bitmez!
