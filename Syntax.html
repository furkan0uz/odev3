JavaScript Sözdizimi

JavaScript sözdizimi, JavaScript programlarının nasıl oluşturulduğu ile ilgili kurallar dizisidir:

        var x, y, z;
        x = 5; y = 6;
        z = x + y;

JavaScript Değerleri

JavaScript sözdizimi iki tür değer tanımlar:
Sabit değerler ve değişken değerler.

Sabit değerlere değişmez değerler denir . Değişken değerlerine değişken denir .

JavaScript Değişmezleri

Sabit değerler yazmak için en önemli kurallar şunlardır:
Sayılar ondalıklı veya ondalıksız olarak yazılır. 

            <script>
            document.getElementById("demo").innerHTML = 10.50;
            </script>

Dizeler çift ​​veya tek tırnak içinde yazılan metindir: 

            <script>
            document.getElementById("demo").innerHTML = 'John Doe';
            </script>

JavaScript Değişkenleri

Bir programlama dilinde, veri değerlerini saklamak için değişkenler kullanılır. JavaScript, değişkenleri bildirmek için var anahtar kelimeyi kullanır . Bir eşittir işareti kullanılır atama değerlere değişkenlere.

Bu örnekte, x bir değişken olarak tanımlanmıştır. Sonra, x değeri 6'ya atanır (verilir):

            <script>
            var x;
            x = 6;
            document.getElementById("demo").innerHTML = x;
            </script>

JavaScript Yorumları

Tüm JavaScript ifadeleri "yürütülmez". Çift bölü sonra Kod //veya aralarında /*ve */bir olarak kabul edilir comment . Yorumlar yok sayılır ve yürütülmez:

            <script>
            var x;
            x = 5;
            // x = 6; I will not be executed  (Sadece Yorum Satırı)        
            document.getElementById("demo").innerHTML = x;
            </script>

JavaScript Tanımlayıcıları

Tanımlayıcılar isimlerdir. JavaScript'te tanımlayıcılar değişkenleri (ve anahtar sözcükleri, işlevleri ve etiketleri) adlandırmak için kullanılır. Yasal adlar için kurallar çoğu programlama dilinde aynıdır. JavaScript'te ilk karakter bir harf veya alt çizgi (_) veya dolar işareti ($) olmalıdır. Sonraki karakterler harf, rakam, alt çizgi veya dolar işareti olabilir.
