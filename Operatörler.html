JavaScript Operatörleri

Değişkenlere değerler atayın ve bunları birlikte ekleyin:

            var x = 5;         // assign the value 5 to x
            var y = 2;         // assign the value 2 to y
            var z = x + y;     // assign the value 7 to z (x + y)

Atama operatörü (=) bir değişkene bir değer atar.

            <script>
            var x = 10;
            document.getElementById("demo").innerHTML = x;
            </script>

İlave operatörü (+) numaralarını ekler.

            <script>
            var x = 5;
            var y = 2;
            var z = x + y;
            document.getElementById("demo").innerHTML = z;
            </script>
            ekran çıktısı= 7

Çarpma operatörü (*) sayıları çarpar.

            <script>
            var x = 5;
            var y = 2;
            var z = x * y;
            document.getElementById("demo").innerHTML = z;
            </script>
            ekran çıktısı= 10

JavaScript Atama İşleçleri

Atama işleçleri, JavaScript değişkenlerine değerler atar.

+= x +=y x=x+y
-= x -=y x=x-y
*= x *=y x=x*y
/= x /=y x=x/y
%= x %=y x=x%y
